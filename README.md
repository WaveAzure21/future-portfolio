# Ibnu.Studio — Portfolio

## 🚀 Quick Start

1. Install dependencies:
```
npm install
```

2. Run the dev server:
```
npm start
```

Opens at **http://localhost:3000**

## 📦 Build for production
```
npm run build
```

## ✨ Features
- Warp hyperspace intro animation
- Warp animation on every nav click
- Custom cursor with magnetic effect
- Glitch text effect
- Typewriter role titles
- 3D tilt project cards
- Scroll-triggered reveal animations
- Animated skill bars
- Video card hover previews
- Particle background
- Fully responsive
